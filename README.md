# 日历记事应用

一个简洁优雅的日历任务管理应用，采用Apple Minimalist设计风格。

## 功能特性

### 核心功能
- ✅ **任务添加**: 支持添加任务标题、描述、完成日期和时间
- ✅ **任务管理**: 编辑、删除任务
- ✅ **状态管理**: 支持待处理、进行中、已完成三种状态
- ✅ **进度跟踪**: 可视化进度条，支持0-100%进度更新
- ✅ **优先级设置**: 低、中、高三种优先级
- ✅ **分类管理**: 工作、个人、学习、其他四种分类

### 视图模式
- 📅 **日历视图**: 直观的月历显示，任务以彩色圆点标记
- 📋 **列表视图**: 详细的任务列表，支持筛选

### 筛选功能
- 全部任务
- 进行中任务
- 已完成任务
- 已逾期任务

### 数据持久化
- 使用LocalStorage本地存储
- 数据自动保存
- 刷新页面数据不丢失

## 技术栈

- **前端**: 纯HTML + CSS + JavaScript
- **设计风格**: Apple Minimalist
- **数据存储**: LocalStorage
- **响应式**: 支持移动端和桌面端

## 使用方法

### 1. 打开应用
直接在浏览器中打开 `index.html` 文件

### 2. 添加任务
- 点击右上角"添加新任务"按钮
- 填写任务信息（标题、描述、日期、时间等）
- 点击"保存任务"

### 3. 查看任务
- **日历视图**: 点击日期查看当天任务
- **列表视图**: 查看所有任务列表

### 4. 更新任务状态
- 点击任务打开详情面板
- 使用进度滑块或快捷按钮更新进度
- 使用状态按钮快速更新状态

### 5. 筛选任务
- 使用侧边栏筛选按钮
- 按状态筛选任务

## 数据结构

每个任务包含以下字段:
```javascript
{
  id: String,           // 唯一标识
  title: String,        // 任务标题
  description: String,  // 任务描述
  date: String,         // 完成日期 (YYYY-MM-DD)
  time: String,         // 完成时间 (HH:MM)
  priority: String,     // 优先级 (low/medium/high)
  category: String,     // 分类 (work/personal/study/other)
  status: String,       // 状态 (pending/in-progress/completed)
  progress: Number,     // 进度 (0-100)
  createdAt: String,    // 创建时间
  updatedAt: String     // 更新时间
}
```
